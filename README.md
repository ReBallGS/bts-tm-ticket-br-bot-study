# Ticket Automation Bot (Study Project)

## 🇺🇸 English

⚠️ This project is for educational purposes only.

This script demonstrates browser automation techniques such as:

* DOM interaction
* Network request interception
* Dynamic element detection
* Adaptive refresh strategies

### Features

* Automatic ticket detection
* Seat availability monitoring
* Smart retry system with random delays
* "Turbo mode" when tickets are found
* Network interception (fetch/XHR)

### Tech Stack

* JavaScript
* Tampermonkey
* Browser APIs

### How to use

1. Install Tampermonkey
2. Add the script
3. Open a supported event page
4. Let the bot monitor availability

### Disclaimer

This project is not intended to bypass platform protections or violate terms of service.
